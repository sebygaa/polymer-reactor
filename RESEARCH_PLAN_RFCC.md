# RFCC 위기대응 연구: 호르무즈 해협 봉쇄 시나리오에 대한 AI 기반 운전조건 최적화

## 1. 연구 배경 및 목적

한국 정유산업은 원유 도입량의 약 70%를 호르무즈 해협을 경유하는 중동산 원유에
의존한다. 해협 봉쇄와 같은 지정학적 위기가 발생하면 정유사는 다음 세 가지
충격을 동시에 겪는다.

1. **원료 가격 충격** — 원유/잔사유 가격 급등 (+60~80%)
2. **원료 품질 변동** — 중동산 공급 차질로 비축 중질유(고 CCR) 또는
   고가의 대체 경질유(저 CCR)로 블렌드가 급변
3. **원료 공급량 제약** — 가용 원료가 설계 처리량 대비 65~85%로 감소

잔사유 유동접촉분해(RFCC)는 정유 마진의 핵심 공정이지만, 원료 품질(특히
Conradson Carbon Residue, CCR)에 대한 민감도가 크다. 중질 원료는 코크 생성을
증가시켜 **재생기 온도 상한(730°C), 공기 블로어 용량, 습식가스 압축기 용량**
같은 설비 제약에 부딪히게 하며, 설계 운전조건을 고수하면 강제 감량 운전과
마진 손실로 이어진다.

**연구 목적**: RFCC의 물리 기반 시뮬레이터를 구축하고, 호르무즈 위기
시나리오에서 (a) 기존 설계 운전("design-book" + 운전원 대응)의 한계를
정량화한 뒤, (b) AI(신경망 대리모델 + 강화학습/최적화)가 운전조건
(라이저 출구온도, 원료 예열온도, 촉매 냉각기 부하, 처리량)을 능동적으로
조정했을 때의 마진 회복 효과를 정량 평가한다.

## 3. 물리 모델 (`rfcc_model.py`)

### 3.1 라이저 — 1차원 플러그 흐름, 단열, 5-lump 반응속도론

Lump: HS(중질원료/슬러리), LCO, 가솔린, 가스(LPG+건성가스), 코크.

반응망 (원료 분해 2차, 2차 분해 1차 반응):

```
HS   → LCO, GASO, GAS, COKE     (k1l, k1g, k1s, k1c)
LCO  → GASO, GAS, COKE          (k2g, k2s, k2c)
GASO → GAS,  COKE  (과분해)      (k3s, k3c)
```

- Arrhenius: `k(T) = k_ref·exp(−Ea/R·(1/T − 1/T_ref))`, T_ref = 520°C
- 촉매 비활성화: `φ(t) = exp(−α·t)`, `α = α₀(1 + 0.06·CCR)` (금속/CCR 영향)
- 부가 코크(additive coke): 라이저 입구에서 `0.85·CCR/100` 즉시 침적
- 반응속도 ∝ C/O (촉매 농도), 흡열 분해열로 라이저 온도 강하 계산
- 고정 스텝 RK4 (30 스텝, 체류시간 3 s), 풀이당 ~0.4 ms

속도상수는 열수지 기준 케이스(ROT 520°C, 예열 220°C, 냉각기 50%, CCR 4)에서
전형적 RFCC 수율(전환율 0.74, 가솔린 0.48, 코크 0.097, C/O 9.8,
T_rgn 668°C)을 재현하도록 튜닝하였다.

### 3.2 라이저 입구 혼합온도 & C/O 결정

재생촉매/원료/분산스팀 엔탈피 수지로 혼합온도를 닫힌 형태로 계산하고,
**라이저 출구온도(ROT)가 설정값이 되도록 C/O를 시컨트 반복으로 풀이**한다
(슬라이드밸브 온도제어 모사). C/O ∈ [3, 14] 범위를 벗어나면 핀(pin) 플래그.

### 3.3 재생기 열수지

```
y_coke·ΔH_burn·(1−손실) − Q_cooler = C/O·Cp_cat·(T_rgn − T_riser_out)
                                    + y_coke·AFR·Cp_air·(T_rgn − T_air)
```

코크 수율이 재생기 온도를 결정하고, 재생기 온도가 다시 필요 C/O에
피드백되는 **FCC 열수지 자기일관 풀이**(감쇠 고정점 반복, ~50 ms/점).
촉매 냉각기(최대 80 MW)는 잔사유 FCC의 핵심 열제거 수단으로 조작변수에 포함.

### 3.4 설비 제약 (위기 시 "벽")

- 재생기 온도 ≤ 730°C (야금 한계)
- 공기 블로어 ≤ 설계의 120%
- 습식가스 압축기 ≤ 설계의 115%
- C/O ∈ [3, 14] (슬라이드밸브)

### 3.5 경제성

마진($/t 원료) = 제품가치(가솔린, LCO, LPG, 건성가스, 슬러리) − 원료비
− 변동운영비 − 촉매보충비(CCR 의존, 금속 피독). 일일 마진(k$/day)으로 환산.

## 5. 호르무즈 위기 시나리오 (150일)

| 국면 | 기간 | 내용 |
|------|------|------|
| 0 정상 | d<15 | 중동산 블렌드 CCR 4%, 정상 가격 |
| 1 봉쇄 충격 | 15–35 | 공급 100→65%, 원료비 ×1.75, 비축 중질유 투입 CCR→6.8, 가솔린 ×1.35 / 경유 ×1.5 |
| 2 적응 | 35–70 | 고가 경질 대체유(미국/서아프리카) 도입 CCR→2.5, 공급 85%, 원료비 ×1.6 |
| 3 부분 재개 | 70–110 | 호송 선단, CCR→4.5, 공급 95%, 가격 완화 |
| 4 정상화 | d≥110 | 기준 복귀 |

AR(1) 노이즈로 일별 변동성 부여. 핵심 물리: **국면 1은 열수지 과잉(재생기
과열) 위기, 국면 2는 열수지 부족(저코크) 위기**로, 서로 반대 방향의 운전
조정이 필요하다 — 단일 규칙 기반 대응이 어려운 이유.

## 6. 기준선(Baseline) 운전 정책

설계 운전점(ROT 520°C, 예열 220°C, 냉각기 50%) 고수 + 운전원의 사후
비상조치 모사:

1. 재생기 과열 → 냉각기 증가 → ROT 3°C씩 감온
2. 재생기 저온(C/O 상한 핀) → 냉각기 감소 → 예열 10°C씩 증가
3. 블로어/압축기 제약 → 처리량 2%씩 감량

이 "사후 대응" 정책이 비교 기준이며, AI는 같은 안전 트림 계층 위에서
**사전적(proactive)** 으로 운전점 자체를 옮긴다.

## 7. AI 방법론

### 7.1 대리모델 (Step 4)
- 입력 5차원: [ROT, 예열, 냉각기, CCR, 처리량] — Latin Hypercube 4,200점
- 출력 8차원: [수율 5개, T_rgn, C/O, 라이저 출구 실온도]
- 순수 NumPy MLP (5→96→96→8, tanh, Adam), 물리 시뮬레이터 대비 ~1000배 가속
- 경제성/제약은 대리모델 출력으로부터 해석적으로 계산

### 7.2 강화학습 (Step 5) — 정책 기반
- 상태 5차원: [CCR, 공급량, 원료가, 가솔린가, LCO가] (정규화)
- 액션 4차원: [ROT, 예열, 냉각기, 처리량] (tanh-squash, 설계점 근방 초기화)
- 알고리즘: REINFORCE + Adam (저장소의 LDPE RL 연구와 동일 계열)
- **분산 감소 — 쌍대조(paired/antithetic) 샘플링**: 같은 상태에 두 액션을
  샘플링하고 보상 차이를 advantage로 사용. 상태(시황) 간 보상 분산이
  완전히 상쇄되어, 전역 배치 베이스라인 대비 학습이 극적으로 안정화됨
  (본 연구에서 전역 베이스라인은 진동/발산, 쌍대조는 단조 수렴)
- 탐색 노이즈 스케줄 σ: 0.30 → 0.05, advantage 클리핑 ±3
- 보상: 일일마진(k$/day) − 제약위반 페널티(T_rgn 초과 °C, 블로어/압축기 초과율,
  ROT 설정값 유지 불능)
- 대리모델 환경에서 도메인 랜덤화 학습 → 진짜 시뮬레이터에서 검증 배치

### 7.3 대리모델 직접최적화 (Step 5) — 디지털트윈 검증형 MPC
- 매일의 (원료, 가격) 관측에 대해 대리모델 위에서 무작위탐색(LHS 6,000점)
  + Nelder-Mead 정련으로 운전점 제안
- **검증 단계**: 제안된 운전점을 물리모델(디지털트윈)로 설계 운전점과 비교,
  우월할 때만 채택 — "대리모델이 제안하고 정밀모델이 검증"하는 산업 표준
  surrogate-RTO 워크플로

### 7.4 배치(Deployment) 평가
세 정책(기준선 / RL / 대리모델-MPC)을 동일 시나리오·동일 안전 트림 하에
**물리 시뮬레이터로** 150일 평가. 지표: 일일·누적 마진, 제약 위반,
처리량 손실, 국면별 운전조건 변화.

## 8. 기대 결과 및 가설

1. 국면 1(중질유): AI는 냉각기 선제 최대화 + ROT/처리량 동시 최적화로
   기준선의 수동적 감량 대비 마진 손실을 줄인다.
2. 국면 2(경질유 + 고제품가): AI는 ROT 상향(과분해 한계까지)과 예열 조정으로
   고마진 기회를 포착한다 — 기준선은 설계점에 머물러 기회 손실.
3. 위기 150일 누적으로 AI 정책이 기준선 대비 수 $M 이상의 추가 마진 확보.

## 9. 모델 한계 및 확장 방향

- 라이저 정상상태 가정(일 단위 의사결정이므로 타당), 사이클론/스트리퍼 미모사
- 5-lump 고정 선택도 — 원료 조성(방향족도) 영향은 CCR로 대표
- 가격/공급 시나리오는 가상의 스트레스 테스트이며 예측이 아님
- 확장: 동적(분 단위) 전환 시뮬레이션, 안전계층 포함 제약 RL(CPO/Lagrangian),
  다기간 재고/블렌딩 최적화 결합, 실제 운전데이터 기반 모델 보정
