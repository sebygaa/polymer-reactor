# 결과 보고서 — RFCC 호르무즈 위기 대응 AI 운전 최적화

연구 설계는 `RESEARCH_PLAN_RFCC.md` 참조. 본 문서는 실행 결과 요약이다.
모든 수치는 코드를 그대로 실행해 재현 가능하다 (실행 순서:
`rfcc_step1` → `rfcc_step2` → `rfcc_step3` → `rfcc_step4` → `rfcc_step5`).

## 1. 기준 케이스 (Step 1–2)

열수지로 닫힌 RFCC 유닛 (ROT 520°C, 예열 220°C, 냉각기 50%, CCR 4 wt%,
설계 처리량 7,300 t/d):

| 항목 | 값 | 항목 | 값 |
|---|---|---|---|
| 전환율 | 0.740 | C/O | 9.8 |
| 가솔린 수율 | 0.477 | 재생기 온도 | 668 °C |
| LCO 수율 | 0.145 | 혼합 온도 | 553 °C |
| 가스 수율 | 0.166 | 마진 | 75.2 $/t |
| 코크 수율 | 0.097 | 일일 마진 | 549 k$/day |

핵심 검증 (그림 `rfcc_step1_*.png`, `rfcc_step2_*.png`):

- 가솔린 과분해 최적점: 고온일수록 최적 체류시간이 짧아짐 (전형적 FCC 거동)
- **중질 원료 압박(heavy-feed squeeze)**: CCR 4→8 wt%에서 코크 0.097→0.111,
  재생기 온도 668→904°C(설계 운전 유지 시), 전환율 0.74→0.61, 마진 75→34 $/t
- 운전영역 지도: 고예열 영역이 T_rgn≤730°C 제약으로 차단되며, 원료가
  중질화되면 실현가능 영역과 최적점이 함께 이동 → 고정 운전점의 한계

## 2. 호르무즈 위기 시나리오와 기준선 운전 (Step 3)

150일 시나리오(봉쇄 충격 → 적응 → 부분 재개 → 정상화)에서 설계 운전점 +
운전원 비상조치(반응적 트림) 기준선:

| 국면 | 평균 마진 [k$/day] | 비고 |
|---|---|---|
| 0 정상 (d<15) | +590 | |
| 1 봉쇄 충격 (15–35) | **−98** | CCR 6.8, 공급 65%, 원료비 ×1.75 |
| 2 적응 (35–70) | +236 | 고가 경질 대체유 |
| 3 부분 재개 (70–110) | +192 | |
| 4 정상화 (d≥110) | +391 | |

- 150일 누적 마진 38.4 M$ vs 무위기 가상 시나리오 82.3 M$
  → **위기 비용 = 43.9 M$**
- 충격기에 재생기 온도가 한계(730°C)에 반복 충돌, 비상 트림으로 간신히 유지

## 3. AI 대리모델 (Step 4)

- 데이터: 물리모델 LHS 4,200점 (운전 4변수 + CCR)
- 순수 NumPy MLP (5→96→96→8): 시험 R² — 전 출력 ≥ 0.997
  (가솔린 0.9996, 코크 0.9998, T_rgn 0.9999, C/O 0.9999)
- 속도: 44.7 ms/점 → 0.0014 ms/점 (**약 33,000배 가속**) — RL 학습과
  일일 최적화를 실시간급으로 가능케 함

## 4. AI 운전 최적화 및 위기 배치 결과 (Step 5)

두 AI 전략 모두 기준선과 동일한 안전 트림 계층 뒤에서, 물리모델로 평가:

| 정책 | 150일 누적 마진 | 기준선 대비 |
|---|---|---|
| 기준선 (설계점 + 운전원 트림) | 38.4 M$ | — |
| **AI-RL** (paired REINFORCE) | 39.7 M$ | **+1.3 M$** |
| **AI-MPC** (대리모델 + 트윈 검증) | 40.6 M$ | **+2.2 M$** |

국면별 평균 마진 [k$/day]:

| 국면 | 기준선 | RL | MPC |
|---|---|---|---|
| 0 정상 | 590 | 580 | 590 |
| **1 봉쇄 충격** | **−98** | **−59** | **−63** |
| 2 적응 | 236 | 245 | 260 |
| 3 부분 재개 | 192 | 200 | 204 |
| 4 정상화 | 391 | 391 | 394 |

### 주요 발견

1. **AI 이득은 위기 국면에 집중된다.** 정상 시황에서는 마진 표면이 평탄해
   설계점이 거의 최적이지만(개선 ≈0), 봉쇄 충격기에는 RL이 일일 손실을
   40% 줄인다(−98 → −59 k$/day). AI의 가치는 "평시 최적화"가 아니라
   **비정상 상황에서의 신속·선제적 재최적화**에 있다.
2. **AI의 대응 패턴**: 충격기(중질유) — 냉각기 선제 최대화 + 처리량
   하한 감량 + 저severity 운전으로 재생기 온도를 한계에서 멀리 유지.
   적응기(경질유 + 고크랙) — 높은 순환비(C/O)와 예열 조정으로 전환율 회복.
   기준선이 한계 온도에 반복 충돌하는 동안 AI는 **제약 여유를 확보하면서**
   더 높은 마진을 냈다 (안전성과 수익성이 동행).
3. **방법론**: (a) REINFORCE에 쌍대조(paired) 샘플링을 도입하자 전역 배치
   베이스라인의 진동/퇴행이 사라지고 단조 수렴(평가 보상 −43 → +235
   k$/day). 시황별 보상 격차가 큰 contextual 문제에서 필수적. (b) 대리모델
   최적해를 물리모델로 검증하는 하이브리드(MPC)가 150일 중 70일에서
   제안을 채택, 나머지는 설계점 유지 — 대리모델 오차에 강건.
4. RL(+1.3M$)은 MPC(+2.2M$)보다 낮지만 추론이 즉각적(μs)이고 검증 풀이가
   필요 없다. MPC는 매일 물리모델 검증 1회가 필요하다.

## 4.5 Step 6 — 반응기/재생기 분리 모델(v02)과 에너지 밸런스 검증

모델을 `RiserReactor`와 `Regenerator` 두 용기로 명시적으로 분리하고
(`rfcc_model_v02.py`), 각 용기에 기준상태(298.15 K) 대비 엔탈피 흐름
에너지 밸런스를 수립·검증했다 (`rfcc_step6_energy_balance_v02.py`).

**v01 대비 일치성** (두-용기 분리가 결과를 바꾸지 않음을 확인):

| 항목 | v01 | v02 | 차이 |
|---|---|---|---|
| 전환율 | 0.7403 | 0.7390 | −0.0013 |
| 코크 수율 | 0.0973 | 0.0971 | −0.0002 |
| C/O | 9.80 | 9.70 | −0.10 |
| T_rgn [°C] | 668.1 | 669.6 | +1.5 |

CCR 1–9 / ROT 490–545 전 스윕에서 최대 편차: 전환율 0.004, T_rgn 6.9°C.

**에너지 밸런스 닫힘** (기준 케이스, kJ/kg 원료): 반응기 IN 7,346.0 =
OUT 7,346.0 (촉매 5,233.5 + 오일 증기 1,609.0 + 스팀 52.0 + 분해열
451.5), 잔차 0.0000%. 재생기 IN 8,294.7 = OUT 8,294.7 (재생촉매
6,815.0 + 연도가스 922.7 + 냉각기 473.4 + 손실 83.6), 잔차 0.0000%.
무작위 300개 운전점 스캔에서도 양 용기 잔차 ≈ 0 (재생기 ≤ 3.6e-12);
자력 연소 불가 클램프 케이스는 열 부족량과 함께 플래그됨.

**재생기 화학량론 업그레이드**: 코크 CH₀.₇, CO₂ 분율 0.60(부분연소),
과잉공기 15% → AFR 12.16 kg/kg, 발열량 28.7 MJ/kg을 생성엔탈피로부터
계산 — v01의 경험적 상수(AFR 14, 29 MJ/kg)를 제일원리에 가깝게 대체하며
기준점 보정 결과가 v01과 자연스럽게 합치.

**방법론적 주의 (정직한 한계 보고)**: 질량·에너지를 같은 RK4로 적분하면
닫힘 잔차는 적분기의 정확한 선형 항등식이 되어, 잔차 0이 이산화 정확도를
증명하지 않는다. 따라서 잔차 감사는 "부기 일관성 검증"으로 해석하고,
정확도는 별도의 스텝 세분화 시험으로 검증했다(운영 n=30에서 T_out 오차
4×10⁻⁴ K, 4차 수렴 확인).
