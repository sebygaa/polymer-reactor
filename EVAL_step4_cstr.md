# Evaluation Report — `step4_cstr_polymerization.py`

**Model:** LDPE Free-Radical Polymerisation — Dynamic CSTR (Autoclave)  
**Method:** Method of Moments (MOM), 11-state scalar ODE system  
**Scope:** Physical meaning, mass/energy balance validity, issues, comparison with PFR step2

---

## 1. Model Overview

### 1.1 Reactor Type: CSTR (Autoclave)

The CSTR (Continuously Stirred Tank Reactor) represents an industrial LDPE **autoclave reactor** — a high-pressure, mechanically agitated vessel assumed to be perfectly mixed at all times. This is the principal physical distinction from the tubular PFR (step2): there is no spatial coordinate, and every molecule anywhere in the reactor sees the same temperature, concentration, and reaction environment.

The governing principle for every species `Cᵢ` is:

```
d[Cᵢ]/dt = ([Cᵢ]_in − [Cᵢ]) / τ + Rᵢ
```

where `τ = V/F` is the **residence time**, `[Cᵢ]_in` is the feed concentration, and `Rᵢ` is the net production rate from reactions. This replaces the advection–reaction structure `v·d[C]/dz = R` of the PFR.

### 1.2 State Vector (11 ODEs)

| Index | Variable | Description |
|-------|----------|-------------|
| 0 | λ₀ | Zeroth moment of living chains — radical molar count (mol/m³) |
| 1 | λ₁ | First moment — total living chain length (dimensionless×mol/m³) |
| 2 | λ₂ | Second moment — controls Mw of live radicals |
| 3 | μ₀ | Zeroth moment of dead chains — dead chain count (mol/m³) |
| 4 | μ₁ | First moment — total dead chain length → Mn via μ₁/μ₀ |
| 5 | μ₂ | Second moment — controls Mw of dead chains → PDI |
| 6 | ini | Initiator [I] (mol/m³) |
| 7 | mono | Monomer [M] (mol/m³) |
| 8 | CTA | Chain-transfer agent [CTA] (mol/m³) |
| 9 | T | Reactor temperature (K) |
| 10 | Tc | Jacket coolant temperature (K) |

Molecular weight averages are recovered from the dead-chain moments at any time:

```
Mn = Mw_mono × μ₁/μ₀       (number-average)
Mw = Mw_mono × μ₂/μ₁       (weight-average)
PDI = Mw / Mn = μ₀μ₂ / μ₁²
```

## 2. Reaction Scheme

Eight reactions are modelled:

| # | Reaction | Rate constant | Role |
|---|----------|---------------|------|
| 1 | I → 2fR•; R• + M → P₁• | kd | Initiator decomposition, radical generation |
| 2 | Pₙ• + M → Pₙ₊₁• | kp | Chain propagation — monomer addition |
| 3 | Pₙ• + M → Dₙ + P₁• | ktrm | Chain transfer to monomer — controls Mn |
| 4 | Pₙ• + Dₘ → Dₙ + Pₘ• | ktrp | Chain transfer to polymer — creates LCB |
| 5 | Pₙ• + Pₘ• → Dₙ₊ₘ | ktc | Termination by combination |
| 6 | Pₙ• + Pₘ• → Dₙ + Dₘ | ktd | Termination by disproportionation |
| 7 | Pₙ• + CTA → Dₙ + R•_CTA | ktr_cta | CTA chain transfer — primary Mn control lever |
| 8 | Pₙ• →^kβ Pₘ• + Dₙ₋ₘ | kbs | β-scission (lumped into K_eff) |

### Effective Transfer Group K_eff

Reactions 3, 7, and 8 are grouped into a single **effective chain-transfer rate**:

```
K_eff = ktrm·[M] + ktr_CTA·[CTA] + kβ     [s⁻¹]
```

All three act kinetically alike: they terminate a growing chain of length n and re-initiate a new chain of length ~1. The units are consistent (all terms are s⁻¹). This grouping follows the reference PDF formulation and is discussed further in §5.2.

## 3. Mass Balance Equations — Validity Assessment

### 3.1 Living Chain Moments

#### λ₀ — Radical count

```
dλ₀/dt = −λ₀/τ + 2f·kd·[I] − (ktc+ktd)·λ₀²
```

**Physical meaning:** Radicals are not fed to the reactor (no inlet term for λ₀). They are generated by initiator decomposition at rate `2f·kd·[I]` and destroyed by bimolecular termination at rate `(ktc+ktd)·λ₀²`. Chain transfer (K_eff) and LCB (ktrp) do **not** appear because they conserve radical count — one radical is consumed and one is created in each event. **✓ Correct.**

#### λ₁ — Total living chain length

```
dλ₁/dt = −λ₁/τ + kp·[M]·λ₀ + K_eff·(λ₀−λ₁) + ktrp·(λ₀·μ₂−λ₁·μ₁) − (ktc+ktd)·λ₀·λ₁
```

Term-by-term derivation:

| Term | Origin | Derivation |
|------|--------|------------|
| −λ₁/τ | Convection out | No radicals in feed |
| +kp·[M]·λ₀ | Propagation | Adding monomer unit to each radical: Σₙ (n+1−n)·[Pₙ•]·kp·[M] = kp·[M]·λ₀ |
| +K_eff·(λ₀−λ₁) | Transfer (all types) | Pₙ•→P₁•: chain length changes from n to 1; Σ(1−n)·rate = λ₀−λ₁ |
| +ktrp·(λ₀·μ₂−λ₁·μ₁) | LCB | Gain: new radical Pₘ• from dead Dₘ → +ktrp·λ₀·Σ(m·m·[Dₘ]) = +ktrp·λ₀·μ₂. Loss: Pₙ•→Dₙ → −ktrp·λ₁·μ₁ |
| −(ktc+ktd)·λ₀·λ₁ | Termination | Each termination event removes a chain of average length λ₁/λ₀ |

**✓ All terms are physically and mathematically correct.**

#### λ₂ — Second moment of living chains

```
dλ₂/dt = −λ₂/τ + kp·[M]·(2λ₁+λ₀) + K_eff·(λ₀−λ₂) + ktrp·(λ₀·μ₃−λ₂·μ₁) − (ktc+ktd)·λ₀·λ₂
```

Key terms:
- `kp·[M]·(2λ₁+λ₀)`: Adding one unit to chain of length n changes n² by (n+1)²−n² = 2n+1; summing gives 2λ₁+λ₀. **✓**
- `K_eff·(λ₀−λ₂)`: Transfer resets chain to length 1; change in n² is 1−n²; sum = λ₀−λ₂. **✓**
- `ktrp·(λ₀·μ₃−λ₂·μ₁)`: The new radical Pₘ• from dead Dₘ contributes m² to λ₂; gain = ktrp·λ₀·μ₃ (requires μ₃ closure). **✓**

This equation **requires the μ₃ closure** since LCB regenerates live radicals from the dead-chain length distribution.

### 3.2 Dead Chain Moments

#### μ₀ — Dead chain count

```
dμ₀/dt = −μ₀/τ + K_eff·λ₀ + (0.5·ktc + ktd)·λ₀²
```

| Term | Origin |
|------|--------|
| −μ₀/τ | Polymer flows out; no polymer in feed |
| +K_eff·λ₀ | Each transfer event (monomer, CTA, β-scission) creates one dead chain |
| +0.5·ktc·λ₀² | Combination: two radicals → one dead chain (factor 0.5); total = 0.5·ktc·λ₀² |
| +ktd·λ₀² | Disproportionation: two radicals → two dead chains; total = ktd·λ₀² |

Note: **ktrp does not appear in μ₀** — LCB converts Dₘ to Pₘ• (live) and Pₙ• to Dₙ (dead) simultaneously; the net change in dead chain count is zero. **✓ Correct.**

#### μ₁ — Total dead chain mass

```
dμ₁/dt = −μ₁/τ + K_eff·λ₁ + ktrp·(λ₁·μ₁−λ₀·μ₂) + (ktc+ktd)·λ₀·λ₁
```

LCB term derivation:
- **Gain** (+ktrp·λ₁·μ₁): Pₙ•→Dₙ. Rate = ktrp·n·[Pₙ•]·Σₘ m·[Dₘ] = ktrp·λ₁·μ₁
- **Loss** (−ktrp·λ₀·μ₂): Dₘ→Pₘ•. Rate = ktrp·Σₙ[Pₙ•]·Σₘ m²·[Dₘ] = ktrp·λ₀·μ₂

The attack rate on dead chains is weighted by chain length m (proportional to number of internal C–H bonds), which gives the μ₂ term in the loss expression. **✓ Correct.**

Termination: combination creates Dₙ₊ₘ (contributing n+m), and disproportionation creates Dₙ+Dₘ (also contributing n+m per event pair). Both give (ktc+ktd)·λ₀·λ₁. **✓**

#### μ₂ — Second moment of dead chains (governs Mw and PDI)

```
dμ₂/dt = −μ₂/τ + K_eff·λ₂ + ktd·λ₀·λ₂ + ktc·(λ₀·λ₂+λ₁²) + ktrp·(λ₂·μ₁−λ₀·μ₃)
```

Termination contributions to μ₂:
- **Disproportionation** (n² + m² per event pair): Σ 0.5·ktd·(n²+m²)·[Pₙ•][Pₘ•] = ktd·λ₀·λ₂ **✓**
- **Combination** ((n+m)² = n²+2nm+m² per event): Σ 0.5·ktc·(n+m)²·[Pₙ•][Pₘ•] = ktc·(λ₀·λ₂+λ₁²) **✓**

The different ktc vs. ktd expressions explain why combination increases PDI more than disproportionation: the cross-term 2·ktc·λ₁² broadens the distribution.

### 3.3 Species Balances

#### Initiator

```
d[I]/dt = ([I]_in − [I]) / τ − kd·[I]
```

Standard CSTR balance with first-order decomposition. At steady state: `[I]_ss = [I]_in / (1 + kd·τ)`. **✓**

#### Monomer

```
d[M]/dt = ([M]_in − [M]) / τ − kp·[M]·λ₀
```

**Approximation:** Only propagation (`kp·[M]·λ₀`) consumes monomer in the balance. Chain transfer to monomer (`ktrm·[M]·λ₀`) also consumes one monomer per event, but since Cm_eff = ktrm/kp ≈ 2.5×10⁻⁴, the omitted term is <0.03% of the propagation term. This is the same convention used in step2 (PFR). **Acceptable approximation. ✓**

#### CTA

```
d[CTA]/dt = ([CTA]_in − [CTA]) / τ − ktr_CTA·[CTA]·λ₀
```

CTA is consumed bimolecularly with living radicals. Each transfer to CTA terminates one chain, so CTA acts as a molecular weight regulator — higher `[CTA]_in` → lower Mn. **✓**

## 4. Energy Balance Equations — Validity Assessment

### 4.1 Reactor Temperature

The full energy balance on the reactor volume V is:

```
ρ·Cp·V · dT/dt = ρ·Cp·F·(T_in − T) + (−ΔHp)·Rp·V − UA·(T − Tc)
```

Dividing by ρ·Cp·V:

```
dT/dt = (T_in − T)/τ + (−ΔHp)/(ρ·Cp) · kp·[M]·λ₀ − (UA)/(V·ρ·Cp) · (T − Tc)
```

The code implements exactly this. Term-by-term:

| Term | Code | Physical meaning |
|------|------|-----------------|
| (T_in−T)/τ | `(T_in - T)/tau` | Sensible heat in feed vs. reactor |
| Heat generation | `(-dH_p)/(rho*Cp)*kp*mono*l0` | Exothermic propagation; dH_p = −93 kJ/mol (negative), so −dH_p > 0 |
| Heat removal | `-a_r*(T-Tc)` where `a_r = UA/(V·ρ·Cp)` | Jacket heat exchange |
| Controller | `−max(0, T−T_sp)·15.0` | Proportional controller, K_c = 15 K/s/K |

**Heat source:** Only propagation (`kp·[M]·λ₀`) contributes to heat generation. Termination and initiation are both exothermic but typically contribute <2% of total heat in LDPE and are standard to neglect. **Acceptable approximation. ✓**

**Temperature controller:** The proportional controller activates above T_sp = 240 °C (513.15 K), representing automatic cooling augmentation present in industrial autoclaves. With K_c = 15 s⁻¹, the "steady-state offset" from setpoint is < 1 °C — effectively an upper temperature bound rather than a true feedback controller. **Physically reasonable. ✓**

### 4.2 Jacket Temperature

```
dTc/dt = (F_c/V_c)·(Tc_in − Tc) + [UA/(V_c·ρ_c·Cp_c)]·(T − Tc)
```

Full balance: `ρ_c·Cp_c·V_c·dTc/dt = ρ_c·Cp_c·F_c·(Tc_in − Tc) + UA·(T − Tc)`

Dividing by ρ_c·Cp_c·V_c gives the code expression exactly. **✓**

**Note on flow direction:** The coolant flows co-currently with the reactor in this lumped CSTR model — there is no spatial coordinate for the jacket, so the inlet/outlet distinction collapses to a single well-mixed jacket temperature. This is conservative (less effective than counter-current). For the well-mixed autoclave, this is physically appropriate.

## 5. Issues and Observations

### 5.1 β-Scission Treatment Differs from step2 (PFR)

**In step2_rxn_w_T_effect_v04.py (PFR):**
```python
# Symmetric random-scission: Pₙ → P_{n/2} + D_{n/2}
dλ₁/dt += −kbs·λ₁/2        # live radicals shorten
dλ₂/dt += −kbs·λ₂·2/3
dμ₀/dt += +kbs·λ₀           # one dead fragment per event
dμ₁/dt += +kbs·λ₁/2
dμ₂/dt += +kbs·λ₂/3
```

**In step4_cstr_polymerization.py (CSTR):**
```python
K_eff = ktrm·mono + ktr_cta·cta + kbs   # kbs lumped as unimolecular transfer
```

The CSTR lumps β-scission identically to chain transfer, treating it as `Pₙ•→Dₙ + P₁•` (chain terminated, new radical of length 1 created). This ignores the dead chain fragment (`Dₙ₋ₘ`) that should be produced and over-simplifies the effect on Mn.

**Consequence:** At temperatures where kbs is significant (T > 260 °C), the CSTR model will predict slightly higher Mn than the full symmetric-scission treatment, because it does not add the low-Mn dead fragments to μ₀/μ₁. At the operating range T = 200–250 °C, kbs ≈ 10⁻³–10⁻² s⁻¹ vs. K_eff ≈ ktrm·mono ≈ 4 s⁻¹, so the contribution is < 0.3% and the approximation introduces negligible error.

**Assessment:** Acceptable at T < 250 °C; should be corrected if the model is used above 260 °C.

### 5.2 Missing ktrm·[M]·λ₀ from Monomer Balance

Transfer to monomer (`Pₙ• + M → Dₙ + P₁•`) consumes one monomer molecule per event. The omitted term is `−ktrm·mono·λ₀`. At 200 °C, ktrm/kp ≈ 2.5×10⁻⁴, so the error in monomer consumption is < 0.03%. This is consistent with the PFR (step2) formulation.

**Assessment:** Standard and acceptable approximation. ✓

### 5.3 DPn QSSA Formula — Approximation

The QSSA formula used:
```python
DPn_q = kp_q*mono_in / (Keff_q + (ktc_q+ktd_q)*l0_q + 1./tau_q)
```

The exact QSSA from the λ₁ balance gives:
```
DPn = (kp·[M] + K_eff + ktrp·μ₂/μ₁) / (K_eff + ktrp·μ₁ + (ktc+ktd)·λ₀ + 1/τ)
```

The code neglects the ktrp·μ₂/μ₁ (numerator) and ktrp·μ₁ (denominator) terms. Since ktrp is small, the error is < 1%. The `1/τ` term in the denominator is CSTR-specific and correct — chains that exit before terminating set a minimum chain length floor. **✓ (with acknowledged approximation)**

### 5.5 Multiple Steady States (UA Sweep)

The code correctly implements a UA sweep to reveal potential **thermal multiplicity** (S-shaped T_ss vs. UA curve). This is a well-known CSTR phenomenon:

- At low UA: heat generated > heat removed → high-T steady state (potential runaway)
- At high UA: heat removed dominates → low-T, low-conversion steady state
- Intermediate UA: possible multiple steady states (ignition/extinction behavior)

The UA sweep from 1,000 to 50,000 W/K spans the physically relevant range. The proportional controller limits the upper branch exploration but does not affect the physical correctness of the UA sensitivity analysis. **Correctly implemented.**

### 5.6 Moment Closure (μ₃)

The Hulburt–Katz closure:
```
μ₃ ≈ μ₂·(2μ₀μ₂ − μ₁²) / (μ₀μ₁)     [Gamma distribution assumption]
```
with conservative fallback `μ₃ ≈ μ₂²/μ₁` and `μ₃ = max(H-K, conservative, 0)`.

This closure introduces ~5–10% error in Mw when LCB is significant (ktrp active), but is the standard choice in LDPE MOM literature (Pladis & Kiparissides 1998; Brandolin 1996). **Acceptable. ✓**

Note: The CSTR closure uses scalar Python operations (`if`, `max`), which is correct since the CSTR has only one spatial node. The PFR (step2) uses NumPy vectorized operations (`np.where`, `np.maximum`) for N grid points.

## 6. Comparison: CSTR (step4) vs. PFR (step2)

| Aspect | CSTR (step4) | PFR (step2) |
|--------|-------------|------------|
| Spatial nodes | 1 | N = 100–200 |
| ODEs | 11 | N × 10 |
| Convection | `(C_in − C)/τ` | `v · dC/dz` |
| CTA | ✓ explicit | ✗ absorbed into ktrm |
| β-scission | Lumped into K_eff | Symmetric scission (6 moment terms) |
| Temperature profile | Uniform (well-mixed) | Axial hot-spot |
| PDI (single unit) | 2–5 (mixing broadens MWD) | 1.7–2.1 (one zone) |
| Conversion per pass | 10–30 % | 5–15 % per zone |
| Heat removal | UA/(V·ρ·Cp) · (T−Tc) | h_r · (T−Tc) per unit length |
| Solver approach | 11-ODE Radau, no Jacobian sparsity | N×10 sparse Radau |
| Typical CPU | < 1 s | 1–10 s |

**Why CSTR gives broader MWD (higher PDI):** Perfect mixing means chains initiated early and chains initiated late coexist and exit together. This residence-time distribution of the chains directly broadens the molecular weight distribution. In contrast, the PFR has a narrow residence-time distribution (plug flow), yielding narrower MWD per zone.

## 8. Recommendations

1. **β-scission consistency:** If the CSTR model is to be compared quantitatively against the step2 PFR results at T > 260 °C, implement the full symmetric-scission terms (separate dμ₀/dt += kbs·λ₀, dμ₁/dt += kbs·λ₁/2, etc.) for consistency with step2_v04.

2. **Energy balance completeness:** Add the heat of CTA transfer `(-ΔH_CTA)·ktr_CTA·[CTA]·λ₀` to the energy balance if operating at high CTA concentrations. At cta_in = 10 mol/m³ this is negligible, but at cta_in > 50 mol/m³ it may influence T_ss by 1–2 °C.

3. **Multiple steady-state detection:** The current UA sweep uses a cold-start initial condition, which selects the lower stable branch. To map the upper branch (ignition path), run the sweep with a hot-start initial condition (y0 initialized at high T, high X). This would make the S-curve complete.

4. **CTA transfer constant Cs:** The value `A_ktr_cta = 1e5, Ea_ktr_cta = 32,000 J/mol` gives `Cs = ktr_CTA/kp ≈ 0.07 at 200 °C`. Literature values for propylene (common CTA in LDPE) are Cs ≈ 5×10⁻⁴–3×10⁻³ (Pladis 1998). The current value is 20–140× too high, meaning the model relies heavily on CTA to control Mn. If matching industrial data is intended, A_ktr_cta should be recalibrated.

5. **Monomer balance completeness:** For academic rigor, add `−ktrm·mono·λ₀` to `R_mono`. The effect on computed X is negligible (< 0.03%) but makes the balance formally complete.
